Proyecto Final de Python: Aplicacion que administra el Stock y Cuentas Corrientes de una Farmacia.
-----------------------------------------------------------------------------------------------------------------
La aplicacion contiene los siguientes modelos:

Cliente: datos de los clientes usados para la cuenta corriente.
Proveedor: Datos de los proveedores de insumos, medicamentos, etc.
Producto: Datos de cada producto, presentacion, precio, etc.
Stock: Productos disponibles en stock.
Cuenta: Cada venta en cuenta corriente se carga con los datos de cada producto, cantidad, etc.

Se encuentran en la base de datos "club.db". Todas las tablas poseen datos de prueba.

Cada seccion/pagina tiene su/s formulario/s de busqueda, boton de edicion, delete y de alta.
-----------------------------------------------------------------------------------------------------------------
Para llevar a cabo la prueba suguiero:

*En la pantalla principal, antes de loguearse, hay un link "Video Explicativo" en el menu, donde lo lleva a el video cargado en Youtube.*

1) Para loguearse utilizar el usuario: admin y password: 12345

2) Una vez logueado, navegar por los links a las distintas paginas que componen la aplicacion.

3) En Clientes, puede consultar los datos de clientes ya cargados, dar de alta uno nuevo, editar desde los botones de Acciones o borrarlos. Tambien realizar una busqueda por DNI o nombre.

4) En Cuenta Corriente, puede consultar las ventas hechas, dar de alta uno nuevo items o realizar una busqueda por nombre de producto o nombre de cliente.Tambien es posible hacer una modificacion o delete.

5) En Proveedores puede consultar los proveedores ya cargados, dar de alta uno nuevo o realizar una busqueda por nombre o cuit.Tambien es posible hacer una modificacion o delete.

6) En Stock puede consultar los productos en existencia, dar de alta uno nuevo o realizar una busqueda por ID de Producto.Tambien es posible hacer una modificacion o delete.

7) En Productos puede consultar los productos ya cargados, dar de alta uno nuevo o realizar una busqueda por nombre o droga.Tambien es posible hacer una modificacion o delete.
